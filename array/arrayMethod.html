Array Methods

  • Push => Add the value from end
  • POP => delete from end & return it
  • Unshift => Add to start
  • Shift => delete from start & return it
  • indexOf => return the index
  • include => search for values
  • concat => marge 2 arrays
  • reverse => Reverse the array
  • slice => copies a portion of an array
  • splice => Remove / replace / add elements in place
  • sort => Sorts an array
